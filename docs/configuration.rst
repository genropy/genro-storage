Configuration
=============

Methods
-------

1. Python dicts
2. YAML files  
3. JSON files

Dict Format
-----------

.. code-block:: python

    {'name': 'home', 'type': 'local', 'path': '/path'}

YAML
----

.. code-block:: yaml

    - name: home
      type: local
      path: /path

JSON
----

.. code-block:: json

    [{"name": "home", "type": "local", "path": "/path"}]
